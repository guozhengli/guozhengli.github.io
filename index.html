自动发布系统说明文档

进入文档阅读

  • 自动发布系统

    发布系统-实现了开发环境、测试环境、生产环境的发布。开发合并代码时,自动触发开发环境的应用发布。合并到测试分支时自动构建后等待发布。

  • 资产管理系统

    资产管理-整合了主机列表、资源池、应用列表、机房管理等功能。与发布系统实现联动。发布系统的发布资源都是从资产管理系统中调用使用。

  • 后台管理系统

    后台管理-用户、部门、角色管理

  • 架构说明

    讲解整体发布系统的架构体系

  • # 安装环境
  • pip install -r requirements.txt
  • # 生成数据库
  • python manage.py makemigrations
  • python manage.py migrate
  • # 多库关联时，生成指定数据库表
  • python manage.py migrate --database=数据库名
  • # 初始化环境
  • python manage.py initialize_premission
  • # 启动测试django
  • python manage.py runserver 0.0.0.0:8000
